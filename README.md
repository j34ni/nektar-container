# nektar-container
Dockerfile and Singularity Definition File to build a Nektar++ container

The corresponding image is built on [https://quay.io/](https://quay.io/).

Pull the container image using either:

```
apptainer pull docker://quay.io/jeani/nektar:5.9.0
```

or

```
docker pull quay.io/jeani/nektar:5.9.0
```
